## Running the app

Basic instructions on how to download and run the app

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

 Clone the repo
   ```sh
   git clone https://github.com/M-Harty21/AWS-workshop-rekognition.git
   ```
 Install NPM packages
   ```sh
   npm install or npm i
   ```
 Install Angular
   ```sh
   npm install -g @angular/cli
   ```
 Install AWS SDK
   ```sh
   npm install aws-sdk
   ```
 Install Rekognition Client
   ```sh
   npm install @aws-sdk/client-rekognition
   ```
